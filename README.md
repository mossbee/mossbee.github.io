This is the source code to Hoang-Nhat Nguyen's public academic website: https://mossbee.github.io/. Feel free to clone this code for your own personal use.

**Local preview:** Sections are loaded via JavaScript, so open the site over HTTP (not as a file). From this folder run:
`python3 -m http.server 8000`
then open http://localhost:8000 in your browser.
